# Email Drafter Skill

## Instructions

When asked to draft an email:

1. Ask for (or infer from context): recipient, purpose, tone (formal/casual), key points to cover.
2. Structure the email:
   - **Subject line**: Clear and specific (not vague like "Following up")
   - **Opening**: Context-appropriate greeting
   - **Body**: Logical flow — context → main point → details → ask/CTA
   - **Closing**: Appropriate sign-off
3. Tone guidelines:
   - Formal: No contractions, full sentences, professional vocabulary
   - Casual: Conversational, contractions OK, friendly opener
   - Sales: Benefits-first, clear CTA, urgency without pressure
4. Always offer 2-3 subject line alternatives.
5. If asked to save, use `file_manager` to write to workspace/drafts/.
