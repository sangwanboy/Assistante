# Web Researcher Skill

## Instructions

When asked to research a topic, follow this process:

1. Use the `web_search` tool to search for the most relevant and recent information.
2. Run 2-3 different search queries to get a comprehensive view.
3. Synthesize the information into a clear, structured report with:
   - **Summary**: 2-3 sentence overview
   - **Key Findings**: bullet points of the most important facts
   - **Sources**: list the URLs found
4. Always note the date of your research.
5. If asked to save the research, use the `file_manager` tool to write it to a markdown file in the workspace.
